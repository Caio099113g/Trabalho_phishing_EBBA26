# Trabalho_phishing_EBBA26
O que é Phishing? É um golpe digital onde criminosos usam "iscas" (e-mails ou SMS falsos) para "pescar" dados.  Como funciona na prática:  A Isca: Você recebe um alerta urgente de um banco ou loja.  O Clique: Um link te leva a um site clonado idêntico ao real.  O Roubo: Ao digitar sua senha, os dados vão direto para o golpista.
