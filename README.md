# Trabalho_phishing_EBBA26
O que é Phishing? É um golpe digital onde criminosos usam "iscas" (e-mails ou SMS falsos) para "pescar" dados.  Como funciona na prática: 
1) A Isca: Você recebe um alerta urgente de um banco ou loja.  
2) O Clique: Um link te leva a um site clonado idêntico ao real.
3) O Roubo: Ao digitar sua senha, os dados vão direto para o golpista.
